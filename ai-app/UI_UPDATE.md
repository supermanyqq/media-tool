# UI更新说明

## 语音合成界面更新

本次更新对语音合成（TTS）功能的UI进行了全面改版，参照提供的设计图进行了一比一复刻。

### 主要更新内容

#### 1. TTSPanel（语音合成主面板）
- **布局调整**：改为左右分栏布局
  - 左侧（14栏）：文本输入区域
  - 右侧（10栏）：合成配置区域
  
- **左侧文本输入区**：
  - 标题："输入文字合成语音"
  - 提示文字："输入文字，转换成逼真的语音，赋能场景丰富的实时应用"
  - 大文本框支持输入待合成文本
  - 字符计数器显示当前/最大字符数
  - 底部"开始合成"按钮
  - 合成结果音频播放器和下载功能

- **右侧配置区**：
  - 调用方式选择（模型选择）
  - 音色选择卡片：
    - 未选择时：显示"选择音色"按钮
    - 已选择时：显示音色头像、名称、地区信息
    - 点击可更换音色
  - 语言选择下拉框
  - 底部链接："查看模型 ssml 的更强控制能力"

#### 2. VoiceSelectionModal（音色选择模态框）
- **顶部标签页**：
  - "音色库"
  - "我的收藏"

- **搜索和筛选**：
  - 搜索框：支持按名称、地区搜索
  - 场景分类标签：全部场景、电话客服、消费电子、有声书、短视频配音、陪伴助手、电商直播、语音助手
  - 多维度筛选：
    - 年龄：全部年龄、儿童、青年、中年、老人
    - 性别：全部性别、男声、女声
    - 语言：全部语言、中文、英文、方言、小语种

- **音色卡片展示**：
  - 紧凑的横向卡片布局
  - 每个卡片包含：
    - 音色头像（左侧）
    - 音色名称和地区（中间）
    - 收藏星标（右侧）
  - 选中状态高亮显示（蓝色边框和背景）
  - 悬停效果

- **底部操作按钮**：
  - "取消"按钮
  - "确定"按钮

### 技术实现

#### 新增功能
1. **多维度筛选**：支持按场景、年龄、性别、语言进行组合筛选
2. **收藏功能**：用户可以标记常用音色
3. **选择确认机制**：选择音色后需点击"确定"才生效（而非立即选择）
4. **响应式布局**：使用Ant Design的Grid系统实现响应式

#### 样式优化
1. 使用Tag.CheckableTag实现可选择标签
2. 自定义样式实现选中态视觉反馈
3. 音色卡片采用紧凑的横向布局
4. 添加hover效果提升交互体验

### 模拟数据
为了更好地展示UI效果，在tts.js服务中添加了getMockVoices()函数，提供了包含以下属性的模拟音色数据：
- name: 音色ID
- chineseName: 中文名称
- region: 地区信息
- description: 描述
- gender: 性别（male/female）
- age: 年龄段（儿童/青年/中年/老人）
- language: 语言类型（中文/英文/方言/小语种）
- scene: 适用场景

### 使用说明
1. 在TTS面板右侧点击"选择音色"或已选音色卡片
2. 在弹出的音色选择框中：
   - 使用搜索框快速查找
   - 使用场景标签和筛选条件精确定位
   - 点击音色卡片进行选择
   - 点击星标收藏常用音色
3. 点击"确定"完成选择
4. 在左侧输入文本，点击"开始合成"生成语音

### 兼容性
- 保持了与原有API的兼容性
- 当API请求失败时自动降级到模拟数据
- 支持原有的音色选择和合成流程
